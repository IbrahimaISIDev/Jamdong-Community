# Jamdong Community

**Agence créative 360° • Dakar, Sénégal • Depuis 2015**

Site web moderne et responsive pour l'agence de communication Jamdong Community, spécialisée dans la communication globale pour les marques, institutions et porteurs de projets.

## 🎯 À Propos

Jamdong Community est une agence créative fondée en 2015 qui combine la puissance des médias traditionnels avec la précision du digital et l'impact du terrain. Notre approche 360° couvre tous les angles de votre communication, sans exception.

### Notre Mission

> Nous racontons vos histoires, amplifions votre voix et connectons votre marque aux audiences qui comptent — en Afrique et au-delà.

## 🚀 Fonctionnalités

### Sections Principales

- **Hero** : Accueil percutant avec l'identité de l'agence
- **Identity** : Nos chiffres clés et explication de la communication 360°
- **About** : Présentation détaillée de l'agence et de sa philosophie
- **Services** : Nos 6 pôles de services complets
- **Footer** : Informations de contact et navigation

### Services Proposés

1. **Médias Traditionnels** - Télévision, Radio, Podcast, Affichage, Cinéma
2. **Digital & Influence** - Réseaux sociaux, Influenceurs, Newsletters
3. **Hors Médias & Terrain** - Événements, Concerts, Forums, Pop-up
4. **Nouveaux Supports** - Kakémono numérique, QR code, Street art
5. **Impression / Sérigraphie** - Papier, Signalétique, Textiles, Goodies
6. **Branding & Design** - Logo, Charte Graphique, Identité Visuelle

## 🛠️ Stack Technique

- **Framework** : React 18.3.1 avec TypeScript
- **Build Tool** : Vite 5.4.2
- **Styling** : TailwindCSS 3.4.1 avec PostCSS
- **Icons** : Lucide React
- **Database** : Supabase (intégré)
- **Linting** : ESLint avec configuration TypeScript

## 📦 Installation

### Prérequis

- Node.js (version 18 ou supérieure)
- npm ou yarn

### Étapes d'installation

1. Cloner le dépôt :
```bash
git clone <repository-url>
cd JamdongCOM-main
```

2. Installer les dépendances :
```bash
npm install
```

3. Démarrer le serveur de développement :
```bash
npm run dev
```

4. Ouvrir votre navigateur sur `http://localhost:5173`
